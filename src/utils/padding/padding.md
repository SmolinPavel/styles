# padding

Add paddings per breakpoint.

`olt-u-[sm-|md-|lg-|xl-]padding{Top|Right|Bottom|Left}{1-8}`

```html
<section class="olt-u-marginBottom4 olt-u-padding5 olt-u-border">
  <p class="Paragraph">Some content</p>
</section>
<section class="olt-u-marginBottom4 olt-u-padding5 olt-u-border">
  <p class="Paragraph">Some content</p>
</section>
```
