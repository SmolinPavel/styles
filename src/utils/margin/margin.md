# margin

Add margins per breakpoint.

`olt-u-[xs|sm|lg|xl]-margin{Top|Right|Bottom|Left}{1-5|Auto}`

```html
<section class="olt-u-marginBottom4 olt-u-border">
  <p class="Paragraph">Some content</p>
</section>
<section class="olt-u-marginBottom4 olt-u-border">
  <p class="Paragraph">Some content</p>
</section>
```
